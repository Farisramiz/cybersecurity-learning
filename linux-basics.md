# Linux Basics 🐧

## What is Linux?
Linux is an open-source operating system widely used in cybersecurity, servers, and cloud environments.

## Why Linux for Cybersecurity?
- Most servers run on Linux
- Many security tools are built for Linux
- Better control over the system
- Powerful command line

## Basic Commands

### Navigation
- `pwd` – show current directory
- `ls` – list files and folders
- `cd` – change directory

### File Management
- `mkdir` – create a directory
- `touch` – create a file
- `cp` – copy files
- `mv` – move or rename files
- `rm` – delete files

### System
- `whoami` – show current user
- `uname -a` – system information
- `clear` – clear terminal

## Permissions (Basics)
- `r` = read
- `w` = write
- `x` = execute
